Clients sind für eine Vielzahl von Plattformen verfügbar, einschließlich Android.

Alle Clients, die eine Verbindung zu diesem Server herstellen und sich unter Verwendung derselben Kombination aus Benutzername und Kennwort authentifizieren, werden eine gemeinsame Zwischenablage gemeinsam nutzen. Wenn ein Client neue Daten in die Zwischenablage kopiert, werden diese Daten an den Server gesendet und anschließend an alle authentifizierten Clients gesendet.

Alle Daten über die Zwischenablage werden Ende-zu-Ende verschlüsselt.
