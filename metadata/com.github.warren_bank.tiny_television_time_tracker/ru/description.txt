Функции включают в себя:

● поиск телесериалов по названию
● добавить телесериалы в результаты
&nbsp;&nbsp;поиска в список
● сортировать и фильтровать список
&nbsp;&nbsp;добавленных телесериалов
● детализируй в список, чтобы про читать

&nbsp;&nbsp;ее доступных эпизодах

● пометить ряды как отключенные

&nbsp;&nbsp;&nbsp;&nbsp;смотреть сериалы

&nbsp;&nbsp;&nbsp;&nbsp;эпизодах
