以下の機能があります。

● 名前によるテレビ番組の検索
● 検索結果にテレビ番組をリストに追加する
● 追加されたテレビ番組のリストを並べ替え、フィルタリングする
● 各シリーズおよびその使用可能なエピソードについての詳細情報を参照するために、リストにドリルを入れる。
● 監視されているとして、エピソードを示す
● シリーズに無効のマークを付ける
● 各系列と外部 URL の関連付け
&nbsp;&nbsp;○ ex: シリーズが監視される Web サイト
● リストのテレビシリーズを更新する
&nbsp;&nbsp;○ 新しいエピソードについての情報を得るために

ネットワーク要件:

● 追加されたテレビ番組のリストは、完全にオフラインで使用することができる
● インターネット接続は、テレビシリーズを手動で追加または更新するためにのみ必要である

履修単位 :

● すべてのデータは、 TMDB API のバージョン 3 を使用して、
&nbsp;&nbsp;Movie Database (TMDB)から取得されます。
