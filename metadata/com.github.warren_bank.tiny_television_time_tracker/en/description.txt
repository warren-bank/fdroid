Features include the ability to:

<ul>
<li>search for television series by name</li>
<li>add television series in search results to a list</li>
<li>sort and filter the list of added television series</li>
<li>drill into the list to read detailed information
about each series and its available episodes</li>
<li>mark episodes as having been watched</li>
<li>mark series as disabled</li>
<li>associate each series with external URLs<ul>
<li>ex: web sites where the series can be watched</li>
</ul>
</li>
<li>update the television series in the list<ul>
<li>to obtain information about new episodes</li>
</ul>
</li>
</ul>

Network requirements:

<ul>
<li>the list of added television series
can be used while fully offline</li>
<li>an internet connection is only needed
to manually add or update television series</li>
</ul>
