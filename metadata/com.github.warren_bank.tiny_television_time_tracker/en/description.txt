Features include the ability to:

● search for television series by name
● add television series in search results to
&nbsp;&nbsp;a list
● sort and filter the list of added
&nbsp;&nbsp;television series
● drill into the list to read detailed
&nbsp;&nbsp;information about each series and its
&nbsp;&nbsp;available episodes
● mark episodes as having been watched
● mark series as disabled
● associate each series with external URLs
&nbsp;&nbsp;○ ex: web sites where the series can be
&nbsp;&nbsp;&nbsp;&nbsp;watched
● update the television series in the list
&nbsp;&nbsp;○ to obtain information about new episodes

Network requirements:

● the list of added television series can be
&nbsp;&nbsp;used while fully offline
● an internet connection is only needed to
&nbsp;&nbsp;manually add or update television series
