ഉപയോക്താവിന്റെ പ്രതികരണം:

● ഒരു ടോസ്റ്റ് കാണിക്കുകName
● ഒരു അറിയിപ്പ് കാണിക്കുകName
● ഒരു ഓഡിയോ റിംഗോണ് കളിക്കുകName

എല്ലാ തരത്തിലുള്ള ഉപയോക്താവിന്റെ ഫീഡ്ബേസുകൾ ഇൻടെന്റ് ആരംഭിക്കുന്നു. ഈ ഉപകരണം മറ്റ് ആപ്ലിക്കേഷനുകൾ ഉപയോഗിച്ച്, ഒരു കമാൻഡ് ലൈൻ ടെർമിനൽ എമുലേറ്ററിൽ നിന്നും നേരിട്ട് ഉപയോക്താവിനെ ഉപയോഗിക്കാം.
