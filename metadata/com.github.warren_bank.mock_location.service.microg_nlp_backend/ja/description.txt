"Mock my GPS" が "UnifiedNlp"のモック・ロケーション・プロバイダーとして動作することを許可します。

「 UnifiedNlp 」は、以下のとおりです。

● " マイクロG" の構成要素として一般的にインストールされる
● Google の Android の ROM を使用することを意図した
● 「 Google ロケーション・サービス」のドロップイン置換

「 UnifiedNlp」のモック・ロケーション・データの統合 :

● 「 Mock my GPS」がアクティブである間のみ有効です。
● 他の「UnifiedNlp」バックエンド・プラグインに優先順位を付け、タイム・スタンプ
&nbsp;&nbsp;45 秒を拡張して、最新のものとして表示されるようにします。
