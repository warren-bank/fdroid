Video URLs on the current web page are displayed in a list.

Settings determine the behavior when a video URL is selected:

<ul>
<li>the internal video player includes a <em>Chromecast</em> sender</li>
<li>the external video player allows another app to play the video URL</li>
<li>the <em>&quot;ExoAirPlayer&quot;</em> and <em>&quot;HLS Proxy&quot;</em> senders
each load a static web page SPA client for the chosen server</li>
</ul>
