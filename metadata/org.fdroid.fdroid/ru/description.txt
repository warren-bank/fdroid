F-Droid-это установочный каталог программных приложений для Android. Клиентское приложение F-Droid упрощает просмотр, установку и отслеживание обновлений на вашем устройстве.

Он соединяется с любыми репозиториями, совместимыми с F-Droid. По умолчанию используется файл repo по адресу f-droid.org, который содержит только программное обеспечение bona fide libre.

Android сам по себе открыт в том смысле, что вы можете установить APKs из любой желательной системы, но есть много хороших причин для использования F-Droid в качестве менеджера приложений libre:

● Получать уведомления при наличии
&nbsp;&nbsp;обновлений
● При необходимости можно загрузить и
&nbsp;&nbsp;установить обновления автоматически
● Отслеживать старые и бета-версии
● Отфильтровать приложения,
&nbsp;&nbsp;несовместимые с устройством
● Поиск приложений по категориям и
&nbsp;&nbsp;описаниям с возможностью поиска
● Доступ к связанным URL для
&nbsp;&nbsp;пожертвований, исходным кодом и т.д.
● Оставайтесь в безопасности, проверив
&nbsp;&nbsp;сигнатуры индекса репо и хэши APK
