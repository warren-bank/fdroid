Features include the ability to:

<ul>
<li>configure and save any number of application profiles</li>
<li>each application profile can specify:<ul>
<li>title</li>
<li>file path to main Javascript file</li>
<li>command line options for the Node.js runtime</li>
<li>command line options for the application</li>
<li>environment variables</li>
<li>current working directory</li>
</ul>
</li>
<li>application profiles are grouped by:<ul>
<li><em>&quot;exec&quot;</em><blockquote>

Applications share a single process.<br>
Only one can execute at a time.

</blockquote>
</li>
<li><em>&quot;fork&quot;</em><blockquote>

Each application is given its own process.<br>
Each executes in a foreground service.<br>
An associated notification kills the process.

</blockquote>
</li>
</ul>
</li>
<li>backup and restore:<ul>
<li>export all application profiles to JSON</li>
<li>import new application profiles from JSON</li>
</ul>
</li>
</ul>

Embedded runtime environment:

<ul>
<li>Node.js 12.19.0</li>
<li>OpenSSL 1.1.1g</li>
</ul>
