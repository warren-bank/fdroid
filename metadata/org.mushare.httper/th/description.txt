แอ็ตทริบิวต์คำร้องขอ HTTP ที่สามารถกำหนดค่าได้:

● เมธอด
● ระเบียบการ
● URL
● ส่วนหัว
● พารามิเตอร์สตริงเคียวรี
● กำลังกาย

คุณลักษณะรวมถึงความสามารถในการ:

● ส่งคำร้องขอหนึ่งครั้ง
● ส่งคำร้องขอซ้ำในช่วงเวลาที่กำหนด
● รีโหลดคอนฟิกูเรชันของคำร้องขอก่อนหน้าจากบันทึกประวัติ
● ข้อมูลการตอบสนอง JSON ที่พิมพ์ได้สวย
● แสดงตัวอย่างข้อมูลการตอบสนอง HTML
