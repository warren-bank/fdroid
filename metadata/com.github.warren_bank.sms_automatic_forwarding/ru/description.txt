Условия определяются списком правил. Каждое правило связывает номер телефона получателя с шаблоном, который должен совпадать с отправителем полученного сообщения SMS.

При получении нового сообщения SMS оно пересылается на номера телефонов получателей по каждому правилу, у которого есть соответствующий шаблон.
