背景:

● Android
&nbsp;&nbsp;インテントを使用すると、任意のインストール済みアプリケーションの正確な部分を開き、任意のタイプまたは数量のデータに渡すことができます。
● そのため、インテントは Android ブックマークのようなものです。

以下の機能があります。

● 名前付きブックマークとしてのインテントの構成と保存
● ブックマークの使用先:
&nbsp;&nbsp;○ アクティビティーの開始
&nbsp;&nbsp;○ サービスの開始
&nbsp;&nbsp;○ ブロードキャストを送信する
● 後で使用するブックマークのスケジュール :
&nbsp;&nbsp;○ 一回
&nbsp;&nbsp;○ 一定の間隔で
● 他のアプリケーションによって開始されたインテントのインターセプト :
&nbsp;&nbsp;○ 変更、
&nbsp;&nbsp;○ 保存する
● バックアップおよびリストア:
&nbsp;&nbsp;○ SQLite データベース全体
&nbsp;&nbsp;○ JSON データ・ファイル
&nbsp;&nbsp;○ Netscape Bookmark HTML データ・ファイル
