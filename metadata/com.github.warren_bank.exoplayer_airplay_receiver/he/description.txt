מקשיב למתאם LAN אלחוטית באמצעות HTTP API ללא מדינה (מבוסס על AirPlay v1). פרסום הזמינות עבור גילוי באמצעות mDNS (Bonjour).

תמיכה בהתקנים של כל גדלי התצוגה, תיבות דואר יקרות לא יקרות עם פלט HDMI רזולוציה גבוהה הן פלטפורמת היעד, אך פועלת היטב באותה מידה על ערכות מאוד ישנות עם מסכי רזולוציה נמוכה מאוד.

ה-API הוא נרחב ומתועד במלואו. יישום לקוח של דף אינטרנט סטאטי הופך את פונקציונליות זו לנגישה ביותר. ברמה הבסיסית ביותר, API זה מספק את היכולת "להטיל" כתובות URL של המדיה לתוך רשימת השמעה ושליטה על כל ההיבטים של ההשמעה.
