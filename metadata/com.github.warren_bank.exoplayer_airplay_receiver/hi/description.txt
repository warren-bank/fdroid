एक राज्यहीन HTTP API के उपयोग से वाई-फाई लैन एडाप्टर पर कान लगाए हैं (एयरप्ले v1 पर आधारित). डीएनएस (BDNS) (BDNS) (BDNS) (BDNS) का उपयोग करने के लिए

सभी प्रदर्शन आकार के उपकरणों का समर्थन करें; उच्च रिज़ॉल्यूशन एचडीएमआई आउटपुट के साथ सस्ते जेनेरिक सेट-टॉप बॉक्स लक्ष्य प्लेटफार्म हैं, लेकिन बहुत छोटे कम संकल्प स्क्रीन के साथ बहुत पुराने हैंडसेट पर समान रूप से काम करता है।

एपीआई व्यापक रूप से और पूरी तरह से प्रलेखित है. एक स्थैतिक वेब पेज SPA क्लाइंट कार्यान्वयन इस कार्यक्षमता को बहुत सुलभ बनाता है. अपने सबसे बुनियादी स्तर पर, यह एपीआई मीडिया यूआरएल को प्लेलिस्ट में "कास्ट" करने की क्षमता प्रदान करता है और प्लेबैक के सभी पहलुओं को नियंत्रित करता है.
