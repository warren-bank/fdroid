Прослухають адаптер Wi-Fi LAN, використовуючи безштатний HTTP API (заснований на AirPlay v1). Рекламує свою доступність для виявлення за допомогою mDNS (Bonjour).

Підтримка пристроїв всіх дисплеїв; недорогі загальні коробки set-top, з високою роздільною здатністю HDMI, є цільовою платформою, але працює однаково добре на дуже старих ручних наборах з дуже малими низькороздільними екранами.

Цей інтерфейс є широким і повністю задокументованими. Статична реалізація клієнта SPA робить цю функціональність дуже доступною. На найвищому рівні, цей API забезпечує можливість "викидати" медіа URL в список відтворення і контролювати всі аспекти відтворення.
