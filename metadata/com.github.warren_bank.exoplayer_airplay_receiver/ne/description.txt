Stateless HTTP API (AirPlay v1 मा आधारित) को प्रयोग गरेर वाई-फाई लैन अनुकूलनमा सूचीहरू. MDNS (Bonjour) प्रयोग गरेर खोजको लागि यसको उपलब्धता विज्ञापन गर्दछ.

सबै प्रदर्शन साइजका यन्त्रहरू समर्थन गर्नुहोस्; उच्च रिजोल्युसन HDMI निर्गतका साथ सस्तो जेनेरिक सेट-माथिल्लो बाकसहरू लक्षित प्लेटफर्म हुन्, तर निकै सानो न्यून-रिजोल्युसन स्क्रिन भएको निकै पुरानो ह्यान्डसेटहरूमा पनि समान रूपले काम गर्दछ.

API विस्तृत र पूर्ण रूपले अभिलेखित छ. एउटा स्थिर वेब पेज SPA क्लाइन्ट कार्यान्वयनले यो कार्यक्षमता अत्यधिक सुलभ बनाउँछ. यसको सबैभन्दा आधारभूत स्तर मा, यो API ले "फेस्ट" मिडिया यूआरएललाई प्लेलिष्टमा र प्लेब्याकको सबै पक्षलाई नियन्त्रण गर्न सक्ने क्षमता प्रदान गर्दछ.
