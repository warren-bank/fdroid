Lytter på Wi-Fi LAN-adapteren vha. et statsløst HTTP-API (baseret på AirPlay v1). Tilføjelse af dets tilgængelighed til opdagelse ved brug af mDNS (Bonjour).

API ' et er omfattende og dokumenteret fuldt ud. En statisk webside SPA-klientimplementering gør denne funktionalitet meget tilgængelig. På det mest grundlæggende niveau giver dette API mulighed for at "kaste" medie URL ' er til en liste og styre alle aspekter af afspilning.
