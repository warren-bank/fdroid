Слуша Wi-Fi LAN адаптера с помощта на безплатен HTTP API (базиран на AirPlay v1). Обяви своята наличност за откриване чрез MDNS (Bonjour).

Поддържа устройства от всички размери на дисплея; евтини генерични set-топ кутии с висока резолюция HDMI изход са целевата платформа, но работи еднакво добре на много стари слушалки с много малки екрани с ниска резолюция.

API е обширно и напълно документирано. Статичното приложение за уеб страници на уеб страници прави тази функционалност много достъпна. На най-базовото ниво, този API предоставя възможността за "прехвърляне" на мултимедийни URL адреси в плейлист и контрол на всички аспекти на възпроизвеждането.
