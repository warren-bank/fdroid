Wi-Fi LAN ඇඩැප්ටරය මත ලැයිස්තුගත කිරීම වෙනුවට A stateless HTTP API (AirPlay v1 මත පදනම්). MDNS (Bonjour) භාවිතා කරමින් සොයාගැනීම සඳහා එහි ඇති පැසසුමට අදාළ

සියලු දර්ශක ප්රමාණයේ උපකරණ පරිශිලක; අධි-විභේදන HDMI ප්රතිදාන සමග අමිල සාමාන්ය set-top පෙට්ටි, ඉලක්කගත වේදිකා වේ, නමුත් ඉතා කුඩා අඩු විභේදන දෑ සමග ඉතා පැරණි හැන්ඩ්සෙස් මත සමාන හොඳින් ක්රියා කරයි.

API පුළුල් හා සම්පුර්ණයෙන්ම ලේඛනගත වේ. ස්ථිතික වෙබ් පිටුවක් SPA සේවාදායකයා ක්රියාත්මක කිරීම මෙම ක්රියාකාරිත්වය ඉතා ප්රවේශම් වනවා. එහි බොහෝ මූලික මට්ටමේ, මෙම API "කාස්ට්" මාධ්ය URLs මගින් වාදන ලැයිස්තුවක් බවට පත් කිරීමේ හැකියාව සහ සියලු පැතිකඩ පාලනය කරන ලදී.
