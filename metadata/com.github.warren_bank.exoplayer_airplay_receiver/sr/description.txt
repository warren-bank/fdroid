Слуша Wi-Fi LAN адаптер користећи HTTP API (базиран на Airplay v1). Оглашава се доступношћу за откривање помоћу MDNS (Bonjus).

Подржава уређаје свих величина екрана; јефтине генеричке сет-топ кутије са високом резолуцијом HDMI излаза су циљна платформа, али ради једнако добро на веома старим ручним сетовима са веома малим екранима ниског резолуције.

АПИ је обимна и потпуно документована. Статичка веб страница SPA клијент имплементација чини ову функционалност веома приступачном. На свом најосновнијем нивоу, овај АПИ пружа могућност да "баци" медијске URL-ове у листу нумера и да контролише све аспекте репродукције.
