ステートレス HTTP API を使用して Wi-Fi LAN アダプターを listen します ( AirPlay v1をベースにします )。 広告は、 mDNS (Bonjour ) を使用してディスカバリーに使用できるようにします。

すべてのディスプレイ・サイズのデバイスをサポートします。高解像度 HDMI 出力を備えた安価な汎用セットトップ・ボックスはターゲット・プラットフォームですが、非常に小さな低解像度の画面を持つ非常に古いハンドセットにも同様に機能します。

この API は広範囲にわたる文書化されています。 静的 Web ページ SPA クライアント実装により、この機能は非常にアクセス可能となります。 この API は、最も基本的なレベルで、メディア URL をプレイリストに「キャスト」し、再生のすべての側面を制御する機能を提供します。
