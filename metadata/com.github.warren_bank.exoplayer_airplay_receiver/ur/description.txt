ایک بے وطن HTTP API (AirPlay کے v1 پر مبنی) کا استعمال کرتے ہوئے وائی فائی LAN اڈاپٹر پر سامعین. MDNS (Bonjour) کا استعمال کرتے ہوئے دریافت کے لئے اس کی دستیابی کو مشتہر کریں.

تمام ڈسپلے سائز کے آلات. اعلی قرارداد HDMI پیداوار کے ساتھ inمہنگی کلی خانوں ہدف پلیٹ فارم, لیکن بہت چھوٹی کم کے ساتھ بہت پرانی ہینڈ سیٹس پر یکساں طور پر کام کرتا ہے-قرارداد اسکرینوں.

API وسیع اور مکمل طور پر دستاویزی ہے. ایک مستحکم ویب پیج SPA کلائنٹ عمل درآمد اس فعالیت انتہائی قابل رسائی ہوتا ہے. اس کی سب سے بنیادی سطح پر یہ API 'کاسٹ' میڈیا یو آر ایل کو ایک پلے لسٹ میں ڈالنے کی صلاحیت فراہم کرتا ہے اور پلے بیک کے تمام پہلوؤں کو کنٹرول کرتا ہے.
