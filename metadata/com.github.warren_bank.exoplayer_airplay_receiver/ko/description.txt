무상태 HTTP API (AirPlay v1 기반) 를 사용하여 Wi-Fi LAN 어댑터를 리스팅한다. mDNS (Bonjour) 를 사용하여 감지에 대한 가용성을 광고합니다.

모든 디스플레이 크기의 장치를 지원합니다. 고해상도 HDMI 출력이 있는 저렴한 일반 셋톱박스는 대상 플랫폼이지만 매우 작은 해상도의 화면을 가진 아주 오래된 전화기에서도 마찬가지로 잘 작동합니다.

API는 광범위하고 충분히 문서화되어 있다. 정적 웹 페이지 SPA 클라이언트 구현은 이러한 기능을 고도로 액세스 가능하게 한다. 가장 기본적인 레벨에서, 이 API는 재생 리스트로 미디어 URL을 "캐스트" 하고 재생의 모든 측면을 제어하는 기능을 제공한다.
