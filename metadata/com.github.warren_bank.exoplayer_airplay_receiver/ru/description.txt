Прослушивает адаптер Wi-Fi LAN с помощью API HTTP без сохранения состояния (на основе AirPlay v1). Рекламирует свою доступность для обнаружения с помощью mDNS (Bonjour).

Поддерживает устройства всех размеров дисплея; недорогие универсальные ящики с высоким разрешением HDMI-это целевая платформа, но в равной степени хорошо работает на очень старых рукоятах с очень маленькими экранами с низким разрешением.

API обширный и полностью документирован. Статическая реализация клиента SPA веб-страницы делает эту функциональность очень доступной. На самом базовом уровне этот API обеспечивает возможность "приведения" URL-адресов в список воспроизведения и управление всеми аспектами воспроизведения.
