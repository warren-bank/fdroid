Εμφανίζεται στον προσαρμογέα LAN LAN χρησιμοποιώντας ένα ανώνυμο HTTP API (με βάση το AirPlay v1). Διαφημίζει τη διαθεσιμότητα του για την ανακάλυψη χρησιμοποιώντας mDNS (Bonjour).

Υποστηρίζει συσκευές όλων των μεγεθών οθόνη; μη δαπανηρή γενική set-top κουτιά με υψηλής ανάλυσης εξόδου HDMI είναι η πλατφόρμα στόχου, αλλά λειτουργεί εξίσου καλά σε πολύ παλιά ακουστικό με πολύ μικρές οθόνες χαμηλής ανάλυσης.

Το API είναι εκτεταμένο και πλήρως τεκμηριωμένο. Μια στατική ιστοσελίδα SPA υλοποίηση πελάτη καθιστά αυτή τη λειτουργικότητα πολύ προσβάσιμη. Σε πιο βασικό επίπεδο, αυτό το API παρέχει τη δυνατότητα να "cast" URLs των μέσων ενημέρωσης σε μια λίστα αναπαραγωγής και τον έλεγχο όλων των πτυχών της αναπαραγωγής.
