Wi-Fi LAN ઍડપ્ટર (AirPlay v1 પર આધારિત) ની મદદથી Wi-Fi LAN ઍડપ્ટર પર સાંભળે છે. MDNS (Bonjour) ની મદદથી શોધવા માટે તેની ઉપલબ્ધતા જાહેરાત કરે છે.

બધી ડિસ્પ્લે માપના ઉપકરણો; ઊંચી રીઝોલ્યુશન HDMI આઉટપુટ સાથેની અહેવાલ સામાન્ય સુયોજન-ટોચ બોક્સ લક્ષ્ય પ્લેટફોર્મ છે, પરંતુ ખૂબ જ નાની ઓછી રીઝોલ્યુશન સ્ક્રીનની સાથે ખૂબ જ જૂના હેન્ડસેટ્સ પર કામ કરે છે.

API એ વિસ્તૃત અને સંપૂર્ણ રીતે દસ્તાવેજીકરણ થયેલ છે. સ્ટેટિક વેબ પાનું SPA ક્લાયન્ટ અમલીકરણ આ કાર્યક્ષમતાને અત્યંત સુલભિત બનાવે છે. તેના સૌથી મૂળભૂત સ્તરે, આ API એ "ફેસ્ટ" મીડિયા URLs ને પ્લેલીસ્ટમાં અને પ્લેબેકની બધી પાસાઓને નિયંત્રિત કરવા માટે ક્ષમતા પૂરી પાડે છે.
