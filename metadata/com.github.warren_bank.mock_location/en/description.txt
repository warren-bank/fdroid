Features include the ability to:

<ul>
<li>mock a fixed position</li>
<li>simulate a trip</li>
<li>use a joystick to move relative to a mock fixed position</li>
<li>accept latitude and longitude geo-coordinate pairs from Intents</li>
<li>save latitude and longitude geo-coordinate pairs as named bookmarks</li>
</ul>

Variations:

<ol>
<li>AOSP</li>
<li>AOSP + Google Play Services</li>
<li>AOSP + Huawei Mobile Services</li>
</ol>

Android integration:

<ol>

Settings &gt; System &gt; About phone

</blockquote>
</li>
<li>repeatedly click 7 times:<blockquote>

</blockquote>

</ul>
</li>

</li>

</li>
