Runs in a foreground Service:

<ul>
<li>there is no visible UI</li>
<li>makes the Bluetooth adapter discoverable</li>
<li>starts a Bluetooth Socket server</li>
<li>listens for a connection from a sender</li>
</ul>

When a sender connects:

<ul>
<li>stops listening for a sender connection</li>
<li>plays the audio stream from the connected Socket</li>
</ul>

When a sender disconnects:

<ul>
<li>resumes listening for a new sender connection</li>
</ul>
