Выполняется в интерактивных службе:

● отсутствует видимый пользовательский
&nbsp;&nbsp;интерфейс
● делает обнаружение адаптера Bluetooth
● запускает сервер гнезд Bluetooth
● прослушивает соединение от отправителя

При подключении отправителя:

● прекращает прослушивание соединения с
&nbsp;&nbsp;отправителем
● воспроизводит звуковой поток из
&nbsp;&nbsp;сосового сокета

Когда отправитель отсоединяется:

● возобновляет прием нового соединения с
&nbsp;&nbsp;отправителем
