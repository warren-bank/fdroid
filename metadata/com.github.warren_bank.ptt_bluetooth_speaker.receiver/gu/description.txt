● ત્યાં દેખાતુ UI નથી
● બ્લુટુથ ઍડપ્ટર ને નિષ્ક્રિય બનાવે છે
● બ્લુટુથ સોકેટ સર્વર શરૂ કરે છે
● મોકલનારમાંથી જોડાણ માટે સાંભળે છે
