Zu den Features gehören:

● Automatische Erkennung von
&nbsp;&nbsp;AirPlay-Empfängern im selben Netzwerk
● Strahlfotos von Ihrem Gerät zum
&nbsp;&nbsp;AirPlay
● Streamen Sie Videos und Musik von
&nbsp;&nbsp;Ihrem Gerät zum AirPlay
● Spiegeln Sie den Bildschirm von Ihrem
&nbsp;&nbsp;Gerät zu AirPlay
