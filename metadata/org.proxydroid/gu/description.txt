રુટ પુનરાવે છે!

વિશિષ્ટતાનો સમાવેશ થાય છે:

● પ્રોક્સી સર્વર પ્રોટોકોલ માટે આધાર:
&nbsp;&nbsp;○ HTTP
&nbsp;&nbsp;○ HTTPS
&nbsp;&nbsp;○ SOCKS4
&nbsp;&nbsp;○ SOCKS5
● સત્તાધિકરણ પદ્દતિઓ માટે આધાર:
&nbsp;&nbsp;○ મૂળભૂત
&nbsp;&nbsp;○ NTLM
&nbsp;&nbsp;○ NTLMv2
● પ્રોક્સી સર્વરને દાખલ કરવા માટે આધાર:
&nbsp;&nbsp;○ વૈશ્વિક રીતે, બધા કાર્યક્રમો માટે
&nbsp;&nbsp;○ ફક્ત ચોક્કસ કાર્યક્રમો માટે
● બ્લેકલીસ્ટ માટે આધાર:
&nbsp;&nbsp;○ પ્રોક્સી ને bypass કરવા માટે IP
&nbsp;&nbsp;&nbsp;&nbsp;સરનામાઓ
● DNS પ્રોક્સી માટે આધાર
● ઘણાબધા રૂપરેખાઓ માટે આધાર
● પ્રોક્સી આપોઆપ-રૂપરેખાંકન માટે મૂળભૂત
&nbsp;&nbsp;આધાર (PAC) ફાઇલ
● સાથે ચોક્કસ રૂપરેખાને સાંકળવાની
&nbsp;&nbsp;ક્ષમતા:
&nbsp;&nbsp;○ WiFi SSID
&nbsp;&nbsp;○ મોબાઇલ નેટવર્ક
● પ્રોક્સી પર પ્રોક્સી ને બદલવા માટે
&nbsp;&nbsp;વિજેટ
● રનટાઇમ કાર્યક્ષમતા:
&nbsp;&nbsp;○ થોડી બેટરી વપરાશમાં છે
&nbsp;&nbsp;○ નાનું RAM ને વાપરે છે
