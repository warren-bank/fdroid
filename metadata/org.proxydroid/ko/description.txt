기능 포함:

● 프록시 서버 프로토콜 지원:
&nbsp;&nbsp;○ 비밀로 하다
&nbsp;&nbsp;○ 암페어
&nbsp;&nbsp;○ SOCKS4
&nbsp;&nbsp;○ SOCKS5
● 인증 메소드 지원:
&nbsp;&nbsp;○ 기본적인
&nbsp;&nbsp;○ 날름
&nbsp;&nbsp;○ NTLMv2
● 프록시 서버 사용 지원:
&nbsp;&nbsp;○ 전세계적으로, 모든 애플리케이션에 대해
&nbsp;&nbsp;○ 특정 애플리케이션에 대해서만
● 블랙리스트 지원:
&nbsp;&nbsp;○ 프록시를 생략하기 위한 IP 주소
● DNS 프록시 지원
● 복수 프로파일 지원
● 프록시 자동 구성 (PAC) 파일에 대한 기본 지원
● 특정 프로파일을 다음과 연관시킬 수 있는 기능:
&nbsp;&nbsp;○ WiFi SSID
&nbsp;&nbsp;○ 모바일 네트워크
● 프록시를 켜거나 끄는 위젯
● 런타임 효율성:
&nbsp;&nbsp;○ 적은 배터리 소모
