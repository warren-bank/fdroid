Features include:

<ul>
<li>detection of userscript URLs</li>
<li>automatic userscript updates</li>
<li>an extended Javascript API
to expose Android specific capabilities</li>
<li>the ability to ignore HTTPS certificate errors</li>
</ul>

Limitations:

<ul>
<li>userscripts only run in the top window<ul>
<li>script injection into iframes is possible,
but requires a complicated workaround;
refer to the <em>README</em> for details.</li>
</ul>
</li>
</ul>
