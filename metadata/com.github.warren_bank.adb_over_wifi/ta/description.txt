மூலத்தை நீக்குகிறது!

USB வகை முன்னிருப்பு. இது ஒரு USB கேபிள் மற்றும் USB இயக்கிகளை ஒரு கணினியில் Android சாதனத்தை இணைக்க தேவைப்படுகிறது.

WiFi முறைமை வலைப்பதிவுகளும், எந்த கூடுதல் இயக்கிகளையும் தேவையிட தேவையிட இயலாது. ஏதாவது கணினி "adb" வாடிக்கையாளர் பயன்படுத்த முடியும்.

WiFi முறைமை செயல்படுத்தப்பட்டால், இந்த பயன்பாடு "adb" கட்டளையை ஒரு LAN மேல் ஒரு இணைப்பு கேட்க தேவைப்படுகிறது.
