Απαιτεί ρίζα!

Η λειτουργία USB είναι η προεπιλεγμένη. Απαιτεί ένα καλώδιο USB και τους οδηγούς USB για να συνδέσετε τη συσκευή Android σε έναν υπολογιστή.

Η λειτουργία WiFi είναι ασύρματη και δεν απαιτεί επιπλέον οδηγούς. Οποιοσδήποτε υπολογιστής μπορεί να χρησιμοποιήσει τον πελάτη "adb" για να ζητήσει μια σύνδεση με τη συσκευή Android.

Όταν ενεργοποιηθεί η λειτουργία WiFi, αυτή η εφαρμογή εμφανίζει την εντολή "adb" που είναι απαραίτητη για να ζητηθεί η σύνδεση μέσω LAN.
