USB මාධ් ය පෙරනිමි වේ. එයට USB කේබල් හා USB ධාවකයින් ඇන්ඩ්රොයිඩ් උපාංගය පරිගණකයකට සම්බන්ධ කිරීමට අවශ්ය වේ.

WiFi මාදිලි රැහැන් රහිත හා අමතර රියදුරන් අවශ්ය නොවේ. ඕනෑම පරිගණකයක් "adb" සේවාදායකය මගින් ඇන්ඩ්රොයිඩ් උපාංගයට සම්බන්ධතාවයක් ඉල්ලුම් කිරීමට හැක.

WiFi ආකාරය සක්රීය විට, මෙම යෙදුම "adb" විධානය LAN පුරා එවැනි සම්බන්ධතාවයක් ඉල්ලීම සඳහා අවශ්ය වේ.
