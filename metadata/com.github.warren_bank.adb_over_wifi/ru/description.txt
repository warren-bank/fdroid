Требуется корень!

По умолчанию используется режим USB. Для подключения устройства Android к компьютеру требуется USB-кабель и USB-драйверы.

Режим WiFi является беспроводным и не требует дополнительных драйверов. Любой компьютер может использовать клиент "adb" для запроса соединения с устройством Android.

Если включен режим WiFi, то в этом приложении отображается команда "adb", необходимая для запроса такого соединения по локальной сети.
