ต้องใช้ root!

โหมด USB เป็นค่าดีฟอลต์ ซึ่งต้องใช้สายเคเบิล USB และไดรเวอร์ USB เพื่อเชื่อมต่ออุปกรณ์ Android กับคอมพิวเตอร์

โหมด WiFi คือแบบไร้สายและไม่ต้องใช้ไดร์เวอร์เพิ่มเติมใดๆ คอมพิวเตอร์เครื่องใดๆสามารถใช้ไคลเอ็นต์ "adb" เพื่อร้องขอการเชื่อมต่อกับอุปกรณ์ Android

เมื่อโหมด WiFi ถูกเปิดใช้งานแอ็พนี้จะแสดงคำสั่ง "adb" ที่จำเป็นเพื่อร้องขอการเชื่อมต่อผ่าน LAN
