Изисква корен!

Режим USB е по подразбиране. Той изисква USB кабел и USB драйвери, за да свържете устройството с Android към компютър.

Wi-Fi режим е безжичен и не изисква допълнителни драйвери. Всеки компютър може да използва "adb" клиент, за да поиска връзка с устройството Android.

Когато е активиран режим Wi-Fi, това приложение показва командата "adb", необходима за заявка за такава връзка през LAN.
