Потрібен корінь!

Режим USB є типовим. Для підключення пристрою Android потрібен кабель USB та USB для підключення до комп'ютера.

Режим WiFi бездротовий і не потребує додаткових драйверів. Будь-який комп'ютер може використовувати клієнт "adb" для запиту підключення до пристрою Android.

Коли увімкнено режим WiFi, то це додаток відображає команду "adb", необхідний для того, щоб запитати про такий зв'язок через локальну мережу.
