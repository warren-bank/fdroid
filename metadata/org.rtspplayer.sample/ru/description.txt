Пробная версия VXG RTSP Player, которая содержит искалеенную версию VXG Mobile SDK. Воспроизведение ограничено 2 минутами.

К функциям относятся:

● Поддерживаемые типы потоковой
&nbsp;&nbsp;передачи:
&nbsp;&nbsp;○ RTSP
&nbsp;&nbsp;○ RTP
&nbsp;&nbsp;○ UDP (поддержка потока
&nbsp;&nbsp;&nbsp;&nbsp;многоабонентной доставки)
&nbsp;&nbsp;○ HTTP-HLS
&nbsp;&nbsp;○ RTMP
&nbsp;&nbsp;○ MMS
● Запись с видеокамеры
● Цифровое масштабирование и изменение
&nbsp;&nbsp;изображения
● Миниизображения для живых потоков
● Поддержка списка каналов M3U
● Управление списком простых потоков
&nbsp;&nbsp;○ добавление, удаление и изменение
&nbsp;&nbsp;&nbsp;&nbsp;потоков
● Просмотр видео при просмотре в списке
&nbsp;&nbsp;камер
● Воспроизведение для прямого потока HLS
● Аппаратный декодер с аппаратным
&nbsp;&nbsp;ускорением с оптимизацией процессора
&nbsp;&nbsp;○ высокоскоростной рендеринг для
&nbsp;&nbsp;&nbsp;&nbsp;современной платформы
● Многоядерный декодирование
&nbsp;&nbsp;обеспечивает производительность
&nbsp;&nbsp;двухъядерных устройств
