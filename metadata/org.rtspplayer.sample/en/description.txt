Evaluation trial version of VXG RTSP Player, which contains a crippled version of the VXG Mobile SDK. Playback is limited to 2 minutes.

Features include:

<ul>
<li>Supported types of streaming: RTSP, RTP, UDP (Multicast stream support),HTTP-HLS, RTMP, MMS</li>
<li>Recording from live camera</li>
<li>Digital zoom and picture shifting</li>
<li>Thumbnails for live streams</li>
<li>Support M3U channel list</li>
<li>Easy streams list control – add, delete and modify your streams</li>
<li>Preview video while browsing in camera list</li>
<li>Replay for live HLS stream</li>
<li>Hardware decoder using hardware acceleration with processor optimization – high speed rendering for modern platform</li>
<li>Multi-core decoding provides the dual-core device’s performance</li>
</ul>
