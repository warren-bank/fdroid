หมายเหตุ: รุ่น Basic ของ F-Droid Client มีชุดฟีเจอร์ลดลง (เช่นไม่มีแชร์ใกล้เคียงและไม่มีฟีเจอร์ panic) ซึ่งเป็นเป้าหมายของ Android 13 และสามารถทำการอัพเดตแบบไม่มีการโต้ตอบโดยไม่มีส่วนขยายที่มีสิทธิพิเศษหรือ root

F-Droid เป็นแคตาล็อกที่ติดตั้งได้ของแอพพลิเคชันซอฟต์แวร์ libre สำหรับ Android ไคลเอ็นต์แอ็พ F-Droid ทำให้ง่ายต่อการเรียกดูติดตั้งและติดตามการติดตามการอัพเดตบนอุปกรณ์ของคุณ

ซึ่งเชื่อมต่อกับที่เก็บที่เข้ากันได้กับ F-Droid ใดๆ ดีฟอลต์ repo ถูกโฮสต์ไว้ที่ f-droid.org repo ซึ่งมีเพียงซอฟต์แวร์ libre เท่านั้น

Android เองก็เปิดในความรู้สึกที่ว่าคุณมีอิสระที่จะติดตั้ง APKs จากทุกที่ที่คุณต้องการแต่มีหลายเหตุผลที่ดีสำหรับการใช้ F-Droid เป็นโปรแกรมจัดการซอฟต์แวร์ libre ของคุณ:

● รับการแจ้งเตือนเมื่อมีอัพเดต
● ดาวน์โหลดและติดตั้งการอัพเดตโดยอัตโนมัติ
● ติดตามเวอร์ชันเก่าและเวอร์ชันเบต้า
● ตัวกรองแอ็พที่เข้ากันไม่ได้กับอุปกรณ์
● ค้นหาแอ็พผ่านหมวดหมู่และคำอธิบายที่สามารถค้นหาได้
● เข้าถึง URL
&nbsp;&nbsp;ที่เกี่ยวข้องของเงินบริจาคซอร์สโค้ดเป็นต้น
● ปลอดภัยโดยการตรวจสอบลายเซ็นดัชนี repo
&nbsp;&nbsp;และ APK hashes
