注: F-Droid Client 的 Basic 版本具有缩小的功能集 (例如，没有附近的共享，无恐慌功能 ) 。 它针对 Android 13 ，可以在没有特权扩展或 root用户的情况下执行无人照管更新。

F-Droid 是 Android软件应用的一个可安装目录。 F-Droid 客户端应用程序使您可以轻松浏览，安装和跟踪设备上的更新。

它连接到任何 F-Droid 兼容存储库。 缺省回购协议托管于 f-droid.org ，它仅包含真正的自由软件。

Android 本身是开放式的，您可以从您希望的任何位置安装 APK ，但是使用 F-Droid 作为自由软件应用程序管理器的原因有很多 :

● 更新可用时收到通知
● 可选择自动下载并安装更新
● 跟踪旧版本和测试版
● 过滤掉与设备不兼容的应用程序
● 通过类别和可搜索描述查找应用程序
● 访问与捐款，源代码等相关的 URL 。
● 通过检查 repo 索引签名和 APK 散列来保持安全
