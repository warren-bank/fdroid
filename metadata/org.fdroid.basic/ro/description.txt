F-Droid este un catalog instalabil de aplicații software libre pentru Android. Aplicația client F-Droid face ușor de navigat, instalat și ține evidența actualizărilor de pe dispozitiv.

Se conectează la orice repozitoare compatibile F-Droid. Repo implicit este găzduit la f-droid.org, care conține doar software-ul bona fide libre.

Android în sine este deschis în sensul că sunteți liber să instalați APK-uri de oriunde doriți, dar există multe motive bune pentru utilizarea F-Droid ca manager de aplicații software libre:
