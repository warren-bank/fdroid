Эти форматы типичны для недорогих камер защиты IP.

Функции включают в себя:

● настроить неограниченное число потоков
&nbsp;&nbsp;видео
● импорт потоков видео из файла JSON
● каждый поток может включать URL для
&nbsp;&nbsp;двух источников:
&nbsp;&nbsp;○ низкое разрешение
&nbsp;&nbsp;○ высокое разрешение
● выборочно включать и выключать потоки
&nbsp;&nbsp;в списке
● отображать включенные потоки в виде
&nbsp;&nbsp;списка или в виде таблицы
&nbsp;&nbsp;○ использует низкое разрешение (если
&nbsp;&nbsp;&nbsp;&nbsp;доступно)
● выбрать один поток из списка или
&nbsp;&nbsp;макета сетки
&nbsp;&nbsp;○ отображается в полноэкранном
&nbsp;&nbsp;○ использует высокое разрешение (при
&nbsp;&nbsp;&nbsp;&nbsp;наличии)

Хронология:

Первоначально это приложение было записано для тестирования экспериментальной ветви ExoPlayer. В то время эта ветвь имела открытый запрос, который не был объедин через несколько лет:
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

В конце концов эта ветвь была объединена. Теперь это приложение использует стабильную компоновку ExoPlayer. Что, имхо, делает его менее интересным. Но это может быть полезно.
