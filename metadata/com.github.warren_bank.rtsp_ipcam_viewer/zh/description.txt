这些格式是典型的廉价 IP 安全摄像头。

功能包括 :

● 配置不限数量的视频流
● 从 JSON 文件导入视频流
● 每个流可以包含两个源的 URL :
&nbsp;&nbsp;○ 低分辨率
&nbsp;&nbsp;○ 高分辨率
● 选择性地启用和禁用列表中的流
● 在列表或网格布局中显示已启用的流
&nbsp;&nbsp;○ 使用低分辨率 ( 当可用时 )
● 从列表或网格布局中选择单个流
&nbsp;&nbsp;○ 全屏显示
&nbsp;&nbsp;○ 使用高分辨率 ( 当可用时 )

历史 :

此应用程序最初是为了测试 ExoPlayer 的实验分支而编写的。 当时，这个分支有一个开放的拉动请求，几年之后没有合并 :
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

最终，这个分支被合并。 现在，此应用程序使用稳定的 ExoPlayer 构建。 这不太有趣 但这可能还是有用的。
