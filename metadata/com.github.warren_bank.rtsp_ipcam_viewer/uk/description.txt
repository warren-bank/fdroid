Ці формати є типовими для недорогих камер спостереження IP.

Особливості включають в себе можливість:

● налаштувати необмежену кількість
&nbsp;&nbsp;потоків відео
● Імпортувати потоки відео з файла JSON
● кожен потік може включати адреси URL
&nbsp;&nbsp;для двох джерел:
&nbsp;&nbsp;○ Низька
&nbsp;&nbsp;○ велика роздільна
● вибірково увімкнути і вимкнути потоки
&nbsp;&nbsp;у списку
● показати увімкнені потоки у списку або
&nbsp;&nbsp;компонуванні ґратки
&nbsp;&nbsp;○ використовує низький роздільність
&nbsp;&nbsp;&nbsp;&nbsp;(за наявності)
● обрати один потік зі списку або
&nbsp;&nbsp;компонування ґратки
&nbsp;&nbsp;○ показати у повноекранному режимі
&nbsp;&nbsp;○ використовувати роздільну здатність
&nbsp;&nbsp;&nbsp;&nbsp;(у разі наявності)

Історія:

Ця програма була спочатку написана для тестування експериментальної філії ExoPlayer. У той час ця філія мала відкритий запит, який не був злиті через кілька років:
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

Зрештою, ця гілка була об'єднана. Тепер ця програма використовує стабільну побудову ExoPlayer. Що, імо, робить це менш цікавим. Але це може бути корисним.
