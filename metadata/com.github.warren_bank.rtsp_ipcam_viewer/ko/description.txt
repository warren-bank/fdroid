이러한 형식은 저렴한 IP 보안 카메라의 전형입니다.

기능에는 다음 기능이 포함됩니다.

● 비디오 스트림의 무제한 수 구성
● JSON 파일에서 비디오 스트림 가져오기
● 각 스트림은 두 개의 소스에 대한 URL을 포함할 수 있다.
&nbsp;&nbsp;○ 저해상도
&nbsp;&nbsp;○ 높은 해상도
● 목록에서 스트림을 선택적으로 사용 및 사용 안함
● 목록 또는 눈금 레이아웃에서 사용 가능한 스트림을 표시합니다.
&nbsp;&nbsp;○ 낮은 해상도 (사용 가능한 경우) 를 사용합니다.
● 목록 또는 그리드 레이아웃에서 단일 스트림 선택
&nbsp;&nbsp;○ 전체 화면에 표시하기
&nbsp;&nbsp;○ (사용 가능한 경우) 고해상도를 사용합니다.

역사:

이 애플리케이션은 원래 ExoPlayer의 실험 분기를 테스트하기 위해 작성되었다. 이 분기에는 몇 년 후에 병합되지 않은 열린 풀 요청이 있었습니다.
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

결국, 이 지점은 합병되었다. 이제 이 애플리케이션은 ExoPlayer의 안정적인 빌드를 사용한다. 이임새는, 그것을 덜 흥미롭게 하는 것이다. 그러나 그것은 여전히 유용할 수 있다.
