これらのフォーマットは、安価な IP セキュリティー・カメラの典型です。

以下の機能があります。

● 無制限の数のビデオ・ストリームを構成する
● JSON ファイルからのビデオ・ストリームのインポート
● 各ストリームには、 2 つのソースの URL を含める
&nbsp;&nbsp;○ 低解像度
&nbsp;&nbsp;○ 高解像度
● リスト内のストリームを選択的に有効化および無効化
● リストまたはグリッド・レイアウトのいずれかで使用可能なストリームを表示します
&nbsp;&nbsp;○ 低解像度を使用 ( 使用可能な場合 )
● リストまたはグリッド・レイアウトから単一ストリームを選択します。
&nbsp;&nbsp;○ 画面全体を表示
&nbsp;&nbsp;○ 高解像度 ( 使用可能な場合 ) を使用
