這些格式都是典型的廉價 IP 安全相機。

特性包括下列功能：

● 配置無限數目的視訊串流
● 從 JSON 檔匯入視訊串流
● 每一個串流都可以包含兩個來源的 URL ：
&nbsp;&nbsp;○ 低解析度
&nbsp;&nbsp;○ 高解析度
● 選擇性地啟用和停用清單中的串流
● 在清單或網格佈置中顯示已啟用的串流
&nbsp;&nbsp;○ 使用低解析度（當可用時）
● 從清單或網格佈置中選取單一串流
&nbsp;&nbsp;○ 在全螢幕中顯示
&nbsp;&nbsp;○ 使用高解析度（當可用時）

歷程：

這個應用程式最初是用來測試 ExoPlayer 的實驗性分支。 當時，這個分支有一個開放的拉動請求，幾年後還沒有被合併：
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

最終，這個分支被合併。 現在，這個應用程式使用穩定的 ExoPlayer 建置。 那麼， imho ，讓它變得不那麼有趣了。 但這可能還是有用的
