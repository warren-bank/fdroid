รูปแบบเหล่านี้เป็นกล้องรักษาความปลอดภัยของ IP ที่ไม่แพง

คุณลักษณะรวมถึงความสามารถในการ:

● กำหนดคอนฟิกจำนวนที่ไม่จำกัดของสตรีมวิดีโอ
● อิมพอร์ตสตรีมวิดีโอจากไฟล์ JSON
● แต่ละสตรีมสามารถมี URLs
&nbsp;&nbsp;สำหรับสองแหล่งข้อมูล:
&nbsp;&nbsp;○ ความละเอียดต่ำ
&nbsp;&nbsp;○ ความละเอียดสูง
● เลือกที่จะเปิดใช้งานและปิดใช้งานสตรีมในรายการ
● แสดงสตรีมที่เปิดใช้งานในโครงร่างรายการหรือกริด
&nbsp;&nbsp;○ ใช้ความละเอียดต่ำ (เมื่อมีอยู่)
● เลือกสตรีมเดี่ยวจากโครงร่างรายการหรือโครงร่างกริด
&nbsp;&nbsp;○ แสดงในแบบเต็มหน้าจอ
&nbsp;&nbsp;○ ใช้ความละเอียดสูง (เมื่อพร้อมใช้งาน)
