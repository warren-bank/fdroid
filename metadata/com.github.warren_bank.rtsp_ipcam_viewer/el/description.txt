Ιστορία:

Αυτή η εφαρμογή γράφτηκε αρχικά για να δοκιμάσει ένα πειραματικό παράρτημα του ExoPlayer. Εκείνη την εποχή, αυτό το υποκατάστημα είχε ένα ανοικτό αίτημα έλξης που δεν είχε συγχωνευθεί μετά από πολλά χρόνια:
&nbsp;&nbsp;https://github.com/google/ExoPlayer/pull/3854

Τελικά, αυτό το υποκατάστημα συγχωνεύτηκε. Αυτή η εφαρμογή χρησιμοποιεί μια σταθερή κατασκευή του ExoPlayer. Που το κάνει λιγότερο ενδιαφέρον. Αλλά μπορεί να είναι χρήσιμο.
